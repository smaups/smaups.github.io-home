Weather Research


The purpose of this research was to analyze how latitude effects weather. Specifically, we looked at observable trends by a city's location with respect to the equator. We gathered data from over 500 cities using the OpenWeatherMap API. From the API calls we stored the data for analysis.
After collecting the data and storing it in a dataset we used Matplotlib to visualize our analysis. We plotted different measures of weather with respect to their latitude. The four elements we plotted were cloudiness, humidity, temperature, and windspeed.
This site provides the analysis of the data. Visualizations, explanations, and descriptions for each of the factors are provided.
